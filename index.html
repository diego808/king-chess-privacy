Política de Privacidade – King Chess

Última atualização: junho de 2026

O aplicativo King Chess foi desenvolvido por Diego Monteiro como um aplicativo gratuito para estudo de xadrez. Esta página descreve como o aplicativo lida com informações.

Dados coletados

O King Chess não coleta, armazena nem compartilha dados pessoais dos usuários. Nenhuma informação de identificação pessoal é solicitada ou transmitida.

Armazenamento local

As lições, anotações e progresso do usuário são salvos exclusivamente no dispositivo do próprio usuário. Esses dados não são enviados para nenhum servidor externo.

Serviços de terceiros

O aplicativo utiliza o Firebase Remote Config (Google) apenas para verificar se há uma versão mais recente disponível. Nenhum dado pessoal é enviado nesse processo. Consulte a Política de Privacidade do Google para mais informações.

Permissões

O aplicativo requer acesso à internet somente para a verificação de versão descrita acima.

Crianças

O King Chess não coleta dados de nenhum usuário, incluindo menores de idade.
